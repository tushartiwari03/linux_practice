# Linux System Health Monitoring

A bash-based monitoring solution to track CPU load, memory usage, and disk usage on Linux systems.

## Features
- CPU, memory, disk monitoring
- Threshold-based alerts
- Email notifications
- Cron scheduling
- Log rotation

## Technologies
- Linux
- Bash scripting
- Cron
- Logrotate

## Use Case
Designed for proactive system monitoring and automation, similar to production infrastructure environments.
